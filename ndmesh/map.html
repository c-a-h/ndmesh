Description

This site is a web interface for the Mesh network on campus.

Click the points on the map for more information on a specific router, and a thumbnail if available.

Use the Thumbnails link to view clickable thumbnails that will redirect you to a page with more controls over that particular camera.

All thumbnails are clickable links for further controls over their corresponding camera.

When viewing a video stream, use the arrows on your keyboard to control the camera's pan/tilt feature.
